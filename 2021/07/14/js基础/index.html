变量

变量名可以是字母、下划线、$ 和 数字，但不能以数字开头。 (驼峰)

  1. 关键字: catch、finally、do、throw、void…
  2. 保留字：Abstract、Byte、Char、Debugger、Export、Static…

数据类型

  1. ES5基本数据类型：Undefined，Null，Number，String，Boolean。引用类型：Object。
  2. ES6新增了数据类型Symbol，表示独一无二的值。
  3. typeof 返回的是string，表示的是数据类型。
  4. 转换为string：var a= a.toString(); 或者a=string(a);
  5. 将srting转为number(string有一些number没有的方法)：a=number(a)或者直接 +number(a);
    undefined返回NaN, 空串/空格/Null，返回0;
  6. a++ (a在前)表示原变量的值，自增前的量; ++a 表示新值，即自增后的值;
  7. “+”运算符转换规则
    如果不全是数字则优先转化为字符串 因该是如果有一个字符串或者对象就转换字符串

数组的方法(改变本身的)

  • 1.push()
    接收任意数量的参数,添加到数组末尾,返回数组的最新长度

  • 2.pop()
    删除数组最后一项,同时减少数组的length值,返回被删除的项

  • 3.shift()
    删除数组第一项并返回它,length减1

  • 4.unshift()
    接收任意数量的参数,添加到数组开头,返回数组的最新长度

  • 5.sort()
    默认使用unicode编码排序,故需要传入比较函数作为参数

  • 6.reverse()
    数组取反

数组的方法(不改变本身的)

  • 1.concat()
    将数组合并添加到返回的新数组

  • 2.slice()
    根据给出的参数截胡原数组并返回到新数组

  • 3.splice()
    三个参数: 开始的索引位,要删除的元素数量,要插入的任意个元素
    返回被删除的元素,否则返回空数组

  • 4.join()
    把数组中的所有元素通过指定的分隔符进行分隔后,放入一个字符串。

运算符

  1. ==不比较数据类型(会转换数据类型再比较number),故基本不用
  2. ===既比较数据类型,又比较number值,性能最优
  3. !=,不比较数据类型(会转换->性能不是最优)
  4. !==,既比较数据类型,又比较number值,性能最优
  5. 此处有高级用法待挖掘

循环语句

  • for in 得到的是对象的key, 数组/字符串的下标

var arr = [‘red’, ‘green’, ‘blue’];
for(let item in arr) { console.log(‘for in item’, item)}
/*
for in item 0
for in item 1
for in item 2
*/

  • for of 只能得到键值

for(let item of arr) { console.log(‘for of item’, item)}
/*
for of item red
for of item green
for of item blue
*/

  • 可以使用 for…in 循环遍历键名

var obj = { ‘name’: ‘Jim Green’,’age’: 12}
for(let key in obj) { console.log(‘for in key’, key)}
/*
for in key name
for in key age
*/

  • forEach() 方法对数组的每个元素执行一次提供的函数。总是返回undefined

var array = [‘a’, ‘b’, ‘c’];
array.forEach(function(element) {
console.log(element);
})
//a; b; c;

函数

  • sort()方法:
  1. 默认是升序排序,但注意默认是按照字符编码的顺序进行排序
  2. 如果比较的两值刚好是数字，简单点：比较函数两个参数a和b，返回a-b升序，返回b-a降序

sort(function(a, b) {
return a - b;
});

异步编程

return只能返回同步数据, 异步中获取数据的方法有3种

  • 回调函数(将函数作为参数传递给另一个函数,也就是将函数作为参数进行操作):

let target = “hello Yoimiya”;
function getData(fn) {
setTimeout(() => {
fn(target);
}, 500)
}
let result = getData((d) => {
console.log(d);
});
//维护困难

  • 用promise对象获取数据
    这是es6提供的一个内置对象,new Promise(); //专门解决异步问题

let target = “hello Yoimiya2”;
function getData() {
return new Promise((resolve) => {
setTimeout(() => {
resolve(target);
}, 666);
}); //then的函数参数的形参就是resolve传出来的值
}
let p = getData();
p.then((d) => {
console.log(d);
})

  • 用async函数

let target = “hello Yoimiya3”;
function getData() {
return new Promise((resolve) => {
setTimeout(() => {
resolve(target);
}, 666);
}); //then的函数参数的形参就是resolve传出来的值
}
async function pray() {
let data = await getData();
console.log(data);
}
pray();
//直接就获取了promise中的值,不用then,可以一个个获取
